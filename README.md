# IOT-Kitchen-Safe-System
We created an Iot product prototype to build safe kitchen environment mainly focusing on prevent flames. We are using alert system to send warning message to house owner by Sms , Call , warning alarms. And house owner can see realtime data in https://thingspeak.com .
